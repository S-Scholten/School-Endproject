# Persona Index

Welkom bij **Persona Index**! Deze site geeft je een overzicht van alle Persona 3-persona's. Je kunt ze bekijken, doorzoeken en inspecteren voor meer details. Dit project heb ik gemaakt om mijn skills als software developer te oefenen — en om er iets leuks van te maken!

---

## Features voor Gebruikers

- **Indexpagina**: Een overzicht van alle beschikbare persona's.
- **Zoekfunctie**: Zoek eenvoudig op naam of nummer naar specifieke persona's.
- **Inspectiekaart**: Klik op een persona om uitgebreide informatie te bekijken.

---

##  Features voor Admins

Alleen beschikbaar na inloggen.

- **Persona's toevoegen**: Voeg nieuwe persona's toe aan de database.
- **Persona's wijzigen**: Pas bestaande persona's aan op basis van hun nummer.

## Installatiehandleiding

1. **Download**: Haal alle projectbestanden binnen.
2. **Start XAMPP**: Open XAMPP (versie 3.3.0) en start Apache en MySQL.
3. **Importeer database**: Klik op *Admin* bij MySQL en importeer het bestand `import.sql`.
4. **Host lokaal**: Open je browser en ga naar `http://localhost/`.
5. **Test het project**: Open de hoofdbestand en probeer het uit!
6. **Voor gebruik van de admin features**: Username: `Name`.    Password: `password123`.
